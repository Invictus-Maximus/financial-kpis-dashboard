# 💰 Financial KPIs Dashboard

## Objetivo
Construir un P&L simplificado y analizar la rentabilidad por producto,
segmento y geografía, identificando tendencias de crecimiento QoQ e YoY.


## Dataset
- **Fuente:** Financial Sample — Kaggle
- **Link:** https://www.kaggle.com/datasets/atharvaarya25/financials
- **Registros:** ~700 transacciones · 5 productos · 5 segmentos · 5 países


## KPIs Construidos
| Métrica | Fórmula |
|---------|---------|
| Margen Bruto | Utilidad / Ventas Netas × 100 |
| Tasa de Descuento | Descuentos / Ventas Brutas × 100 |
| Ratio COGS/Ventas | COGS / Ventas Netas × 100 |
| Crecimiento QoQ | Variación % vs. trimestre anterior |
| Crecimiento YoY | Variación % vs. mismo trimestre año anterior |

## Herramientas
`Python` · `Pandas` · `NumPy` · `Matplotlib` · `Power BI`

## Hallazgos Clave
- Margen bruto global: **14.23%**  — por debajo del benchmark del 15%
- Producto más rentable: **[Amarilla]** con **[29.3%]** de margen
- País con mayor crecimiento YoY: **Estados Unidos** (+288.4% en Q1 2013→2014)
- Crecimiento YoY brutal: **+249%** de 2013 Q1 a 2014 Q1
- Segmento que concentra el **[44.22%]** de las ventas: **[Government]**
